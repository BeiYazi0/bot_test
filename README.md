# QQ 频道机器人示例

这是一个 QQ 频道机器人示例项目

渣代码，可以借鉴一下。

## 食用方法

1. 在 bot_test 文件夹下的 config.py 中填入你的 appid 和 token。
2. 返回上级文件夹，双击 run.bat 运行即可
